
# Project 1 - JavaScript Background Color Switcher

A simple web application that allows users to switch the background color by clicking on color buttons. Built with HTML, CSS, and JavaScript.

- CSS

## Getting Started
1. Clone the repository:
    ```sh
    git clone https://github.com/Dhruv9099/JavaScript_Tutorial.git
    ```

2. Navigate to the project directory:
    ```sh
    cd JavaScript_Tutorial/07_Projects/01_color_switch_changer
    ```

3. Open `index.html` in a web browser.
